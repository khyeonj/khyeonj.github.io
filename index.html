{ Overview_

서울탁주는 오래된 역사를 가지고 있는 한국의 탁주 브랜드이나 요즘은 젊은 층을 타겟으로 홍보하고 있습니다. 하지만 서울탁주의 웹페이지는 스큐어모비즘 스타일로 디자인 되어있어 젊은 층을 공략하는 광고와 달리 홈페이지는 올드한 느낌을 주고 있습니다. 또한 flash를 사용하고 있어 메인배너가 뜨지 않는 문제점을 분석하여 요즘 트렌드에 맞는 반응형 웹페이지로 리뉴얼 프로젝트를 진행하였습니다.

{ Keywords

직관적인

높은 사용성

{ Project Goal_

정돈된 레이아웃으로 시선을 분산시키지 않고 집중시키며 다양한 동적 요소를 활용하여 눈길을 끌 수 있도록 하였습니다. 최종적으로 사용성과 흥미를 높여 방문객이 증가되는 것을 목표로 작업을 진행하였습니다.
