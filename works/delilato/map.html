네이버지도API 적용하는 방법

  • 애플리케이션 등록하기
  • 기본정보 입력하기 : 비로그인 오픈 API
  • 웹서비스 URL 등록하기 : 웹 서비스 URL에 등록된 페이지에서만 NAVER 지도 API v3를 사용할 수 있습니다.
  • 이용약관 확인하기
  • 발급받은 클라이언트 아이디 확인하기 : Client ID에서 보여주는 클라이언트 아이디를 이용해 NAVER 지도 API v3를 로드할 수 있습니다.
